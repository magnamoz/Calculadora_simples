Calculadora JS

A Calculadora JS foi desenvolvida para exercitar a prática de HTML, CSS e JavaScript. O Projeto foi desenvolvido sem a utilização de frameworks e também possui alguns elementos do estudo sobre desing responsivo iniciado com a #ImersãoCSS da Alura - Imersão Hipsters.CSS.

Icons: Github feito por Pixel perfect, LinkedIn feito por Freepik e E-mail feito por Iconixar disponíveis em www.flaticon.com
